Vanilla NodeJS Server

This server implementation is a mock api server built using only vanilla NodeJS and JavaScript.

Available Routes

[GET: /] response: index.html

[POST: /] request: json-object, response: (a page showing post data)

[GET: /posts] response: mock post data

[POST: /posts] request: { title, content }, response: mock post object

* All data is saved in memory and is not persistent.
